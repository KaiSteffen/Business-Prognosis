Model **Chronos**.

| Ordner | Inhalt |
|:---|:---|
| [`prophet/`](prophet/) | Prophet-Tutorial: Demo, Hands-On und Musterlösung |
| [`chronos/`](chronos/) | Chronos-Tutorial: Demo, Hands-On und Konzept (Prophet↔Chronos) |

Jeder Ordner hat eine eigene Conda-Umgebung (`ts-tutorial.yml`) und ein eigenes README
mit Setup-Anleitung.
